### 1.2 路由

通过 URL 参数判断新建/编辑模式：

| URL | 模式 | `isNew` |
|-----|------|---------|
| `/config/contract-types/new` | 新建 | `true` |
| `/config/contract-types/:id/edit` | 编辑 | `false` |

### 1.3 核心逻辑

```javascript
// 获取路由参数
const route = useRoute()
const isNew = computed(() => route.params.id === 'new')

// 编辑模式：加载已有数据
onMounted(async () => {
  if (!isNew.value) {
    const detail = await getType(route.params.id)
    if (detail) {
      form.id = detail.id
      form.name = detail.name || ''
      form.stance = detail.stance || '甲方立场'
      form.desc = detail.desc || ''
      form.keywords = Array.isArray(detail.keywords) ? detail.keywords.join(',') : (detail.keywords || '')
    }
  }
})

// 保存
async function save() {
  await saveType({
    id: isNew.value ? undefined : form.id,
    name: form.name, stance: form.stance, desc: form.desc, keywords: form.keywords,
  })
  router.push({ name: 'config' })  // 返回配置中心
}
```

### 1.4 "返回合同类型管理"按钮

点击 `router.push({ name: 'config' })` 返回 ConfigCenter 页面。ConfigCenter 的 `onMounted` 会自动加载列表。
