# 审查点与合同类型页面增加删除功能设计说明书

**项目**：contract-review-system  
**日期**：2026-07-01  
**版本**：v1.0  
**对应计划书**：`docs/planning/删除功能计划书.md`

---

## 一、PointEdit.vue

### 1.1 模板

在底部操作栏（`class="actions"`）中增加删除按钮，`el-button` 的 `v-if` 控制在编辑模式才显示：

```html
<div class="actions">
  <el-button @click="router.push({ name: 'config' })">取消</el-button>
  <el-button v-if="!isNew" type="danger" plain @click="deletePoint">删除该审查点</el-button>
  <el-button type="primary" :loading="saving" @click="save">保存</el-button>
</div>
```

### 1.2 Script

在 `<script setup>` 中增加 `ElMessageBox` 导入和 `deletePoint` 函数：

```javascript
import { ElMessage, ElMessageBox } from 'element-plus'
import { deletePoint as deletePointApi } from '../../services/point.service'

async function deletePoint() {
  await ElMessageBox.confirm(
    `确认删除审查点「${form.name}」？删除后可在数据库中恢复。`,
    '提示',
    { type: 'warning' },
  )
  await deletePointApi(form.id)
  ElMessage.success('已删除')
  router.push({ name: 'config' })
}
```

> `deletePoint` 已在 `point.service.js` 中定义，调用 `DELETE /points/{id}`。后端 `audit_point_store.delete_by_id()` 写 `deleted_at = NOW()`，列表查 `WHERE deleted_at IS NULL`，实现软删除。

## 二、ContractTypeEdit.vue

### 2.1 模板

同样在底部操作栏增加删除按钮：

```html
<div class="actions">
  <el-button @click="backToTypes">取消</el-button>
  <el-button v-if="!isNew" type="danger" plain @click="deleteType">删除该合同类型</el-button>
  <el-button type="primary" :loading="saving" @click="save">保存</el-button>
</div>
```

### 2.2 Script

增加 `ElMessageBox` 和删除函数：

```javascript
import { ElMessage, ElMessageBox } from 'element-plus'
import { deleteType as deleteTypeApi } from '../../services/type.service'

async function deleteType() {
  await ElMessageBox.confirm(
    `确认删除合同类型「${form.name}」？删除后可在数据库中恢复。`,
    '提示',
    { type: 'warning' },
  )
  await deleteTypeApi(form.id)
  ElMessage.success('已删除')
  backToTypes()
}
```

> `deleteType` 已在 `type.service.js` 中定义，调用 `DELETE /contract-types/{id}`。后端 `contract_type_store.delete_by_id()` 写 `deleted_at = NOW()` + 删关联表，列表 `WHERE deleted_at IS NULL`。

## 五、验证

```
[审查点删除]
  编辑已有审查点 → 点击"删除该审查点" → 确认弹窗 → 调用 DELETE /audit-points/{id}
  → 后端写 deleted_at → 返回列表 → 该审查点不再显示

[合同类型删除]
  编辑已有合同类型 → 点击"删除该合同类型" → 确认弹窗 → 调用 DELETE /contract-types/{id}
  → 后端写 deleted_at + 删关联 → 返回列表 → 该合同类型不再显示

[新建模式]
  新建审查点/合同类型 → 删除按钮不显示（v-if="!isNew"）
```
