# ContractTypeEdit 滚动条修复审查意见

**审查日期**：2026-07-02  
**审查范围**：`frontend/src/views/config/ContractTypeEdit.vue` 样式部分

---

## 一、问题诊断

页面只有一个滚动条，左右两栏无法各自独立滚动。

### 根因

CSS flex/grid 布局中，子元素缺少高度约束，导致 `overflow-y: auto` 无法触发。

| 选择器 | 当前 | 问题 |
|--------|------|------|
| `.type-shell` | `min-height: 600px` | 容器可以无限撑高，超出的部分由外层 `.page` 滚动 |
| `.type-aside` | 无 `min-height: 0` | grid 子元素默认 `min-height: auto`，不能收缩到比内容更小 |
| `.type-main` | 无 `min-height: 0; overflow-y: auto` | 右栏内容超出时无法独立滚动 |

### 滚动失效链

```
.type-shell: min-height: 600px（无上限）
  → 内容超出视口 → .page: overflow: auto → 页面级滚动条
  → .type-aside 和 .type-main 均无 overflow → 无法独立滚动
```

---

## 二、修改方案

### 修改 1：`.type-shell` 改为固定视口高度

```css
/* 改前 */
.type-shell {
  …
  min-height: 600px;
  overflow: hidden;
}

/* 改后 */
.type-shell {
  …
  height: calc(100vh - 180px);
  overflow: hidden;
}
```

`100vh - 180px` = 视口高度减去 Topbar（68px）+ `.page` 的 padding（20px + 30px）+ 余量。

### 修改 2：`.type-aside` 增加 min-height 和独立滚动

```css
/* 改前 */
.type-aside {
  display: flex;
  flex-direction: column;
  min-width: 0;
  background: #fbfcff;
  border-right: 0.667px solid #e0e6ef;
}

/* 改后 */
.type-aside {
  display: flex;
  flex-direction: column;
  min-width: 0;
  min-height: 0;
  background: #fbfcff;
  border-right: 0.667px solid #e0e6ef;
  overflow-y: auto;
}
```

`min-height: 0` 是 flex/grid 子元素启用滚动的关键——没有它，浏览器会用内容高度作为最小高度，`overflow-y: auto` 无效。

### 修改 3：`.type-list` 增加 min-height，滚动交给父级

```css
/* 改前 */
.type-list {
  flex: 1;
  display: flex;
  flex-direction: column;
  gap: 8px;
  padding: 12px;
  overflow-y: auto;
}

/* 改后 */
.type-list {
  flex: 1;
  min-height: 0;
  display: flex;
  flex-direction: column;
  gap: 8px;
  padding: 12px;
}
```

`.type-aside` 已经接管了 `overflow-y: auto`，`.type-list` 只需 `min-height: 0` 配合 `flex: 1` 伸缩即可。

### 修改 4：`.type-main` 增加独立滚动

```css
/* 改前 */
.type-main {
  min-width: 0;
  padding: 26px 30px 30px;
}

/* 改后 */
.type-main {
  min-width: 0;
  min-height: 0;
  padding: 26px 30px 30px;
  overflow-y: auto;
}
```

### 修改 5（可选）：`.page` 移除自身滚动

```css
/* 当前 */
.page {
  flex: 1;
  padding: 20px 28px 30px;
  overflow: auto;
}

/* 如果内层已接管滚动，外层可去除 */
.page {
  flex: 1;
  padding: 20px 28px 30px;
  overflow: hidden;
}
```

## 四、验证

- 左侧合同类型超过 10 条时，左侧栏出现纵向滚动条
- 右侧表单内容超出视口时，右侧栏出现纵向滚动条
- 两个滚动条各自独立，互不影响
- 左侧"＋ 新建"按钮固定在底部不随列表滚动（在 `.type-aside` overflow 之外）
